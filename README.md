# Análise de E-commerce — Olist (SQL + Power BI)

## Contexto de negócio
A Olist é um marketplace brasileiro que conecta pequenos lojistas a grandes plataformas de e-commerce. Este projeto analisa mais de 96 mil pedidos realizados entre 2016 e 2018 para responder perguntas estratégicas de negócio sobre receita, produtos e logística.

**Dataset:** [Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce/data?select=olist_orders_dataset.csv)

------

## Perguntas respondidas

1. Qual categoria de produto gera mais receita por mês?
2. Quais estados têm o maior ticket médio?
3. Qual a taxa de cancelamento por categoria?
4. Qual o tempo médio de entrega por região?

## Ferramentas utilizadas
- SQLite — modelagem e queries
- Power BI Desktop — visualização
- Git/GitHub — versionamento

## Como reproduzir
1. Baixe o dataset no [Kaggle](link)
2. Importe os CSVs no SQLite
3. Execute as queries na pasta `sql/` na ordem numérica
4. Abra o dashboard em `dashboard/` no Power BI Desktop
